# Print optimal transport sensitivity indices information

Print optimal transport sensitivity indices information

## Usage

``` r
# S3 method for class 'gsaot_indices'
print(x, ...)
```

## Arguments

- x:

  An object generated by
  [`ot_indices`](https://pietrocipolla.github.io/gsaot/reference/ot_indices.md),
  [`ot_indices_1d`](https://pietrocipolla.github.io/gsaot/reference/ot_indices_1d.md),
  or
  [`ot_indices_wb`](https://pietrocipolla.github.io/gsaot/reference/ot_indices_wb.md).

- ...:

  Further arguments passed to or from other methods.

## Value

The information contained in argument `x`

## Examples

``` r
N <- 1000

mx <- c(1, 1, 1)
Sigmax <- matrix(data = c(1, 0.5, 0.5, 0.5, 1, 0.5, 0.5, 0.5, 1), nrow = 3)

x1 <- rnorm(N)
x2 <- rnorm(N)
x3 <- rnorm(N)

x <- cbind(x1, x2, x3)
x <- mx + x %*% chol(Sigmax)

A <- matrix(data = c(4, -2, 1, 2, 5, -1), nrow = 2, byrow = TRUE)
y <- t(A %*% t(x))

x <- data.frame(x)

M <- 25

# Calculate sensitivity indices
sensitivity_indices <- ot_indices(x, y, M)
print(sensitivity_indices)
#> Method: sinkhorn 
#> 
#> Indices:
#>        X1        X2        X3 
#> 0.5902943 0.6334958 0.2930755 
```
